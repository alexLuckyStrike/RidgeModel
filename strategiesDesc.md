# 5 стратегий обратной задачи

## Суть

Модель PCA+ridge даёт уравнение:

```
ΔPC1 = β_V·dV + β_P·dP + β_R·dR
```

где β_V = 7.48, β_P = 7.79, β_R = −2.54.

Одно уравнение, три неизвестных — бесконечно много решений. Каждая стратегия фиксирует два параметра (или задаёт пропорции между ними), оставляя одно конкретное решение.

Все 5 стратегий дают **одинаковый PC1** для одной и той же зоны, но **разные тренировочные воздействия**.

---

## Параметры нагрузки

| Параметр | Обозначение | Единица | Смысл |
|----------|-------------|---------|-------|
| Объём | V (dV) | кг (КПШ × вес) | Суммарный тоннаж тренировки |
| Интенсивность | P (dP) | % от 1ПМ | Средний вес снаряда относительно максимума |
| Восстановление | R (dR) | минуты | Средняя пауза отдыха между подходами |

dV, dP, dR — **относительные отклонения** от индивидуального baseline спортсмена:
```
V_тренировки = V_baseline × (1 + dV)
```

---

## 1. volume_only (Только объём)

**Что меняется:** только объём (dV). Интенсивность и паузы = baseline.

**Ограничения:** dP = 0, dR = 0

**Формула:** dV = ΔPC1 / β_V

**Пример для Зоны 3 (ΔPC1 = +1.04):**
```
dV = +1.04 / 7.48 = +13.9%
dP = 0
dR = 0
→ Тренировка: объём +13.9% от нормы, интенсивность и паузы без изменений
```

**Педагогический смысл:** Тренер увеличивает количество подходов/повторений, не трогая вес на штанге и паузы. Классический объёмный подход школы Шейко.

**Когда применяется:**
- Базовый период подготовки
- Накопление тренировочного объёма
- Работа на выносливость и технику

---

## 2. pace_only (Только интенсивность)

**Что меняется:** только интенсивность (dP). Объём и паузы = baseline.

**Ограничения:** dV = 0, dR = 0

**Формула:** dP = ΔPC1 / β_P

**Пример для Зоны 3 (ΔPC1 = +1.04):**
```
dV = 0
dP = +1.04 / 7.79 = +13.4%
dR = 0
→ Тренировка: вес на штанге +13.4%, объём и паузы без изменений
```

**Педагогический смысл:** Тренер увеличивает вес снаряда, сохраняя количество подходов. Скоростно-силовая работа. Метод сопряжённых воздействий Симмонса (Westside Barbell).

**Когда применяется:**
- Специальная подготовка
- Развитие максимальной силы
- Предсоревновательный период (работа на пик)

---

## 3. recovery_only (Только восстановление)

**Что меняется:** только паузы отдыха (dR). Объём и интенсивность = baseline.

**Ограничения:** dV = 0, dP = 0

**Формула:** dR = ΔPC1 / β_R

**Пример для Зоны 3 (ΔPC1 = +1.04):**
```
dV = 0
dP = 0
dR = +1.04 / (−2.54) = −40.9%
→ Тренировка: паузы сокращены на 40.9%, объём и вес без изменений
```

**Педагогический смысл:** Тренер сокращает паузы отдыха между подходами, создавая метаболический стресс при том же объёме и весе. Для снижения нагрузки (Зона 1) — наоборот, удлиняет паузы.

**⚠ Ограничение:** Для Зоны 1 требуется dR = +98.4% (удвоение пауз) — это **avoid** (3.7σ, за пределами обучающих данных). **Стратегия recovery_only для Зоны 1 исключена.**

**Когда применяется:**
- Зоны 2, 3, 4 (но не Зона 1)
- Тренировки на выносливость с ограничением отдыха
- Метаболический стресс (кроссфит-стиль)

---

## 4. balanced (Сбалансированная)

**Что меняется:** объём (dV) и интенсивность (dP) пропорционально. Паузы = baseline.

**Ограничения:** dR = 0, dV/dP = β_V/β_P (пропорциональны коэффициентам модели)

**Формула:**
```
dV = ΔPC1 / (β_V + β_P × β_V/β_P) = ΔPC1 / (2 × β_V)  [упрощённо]
dP = dV × β_V / β_P
```

**Пример для Зоны 3 (ΔPC1 = +1.04):**
```
dV = +7.0%
dP = +6.7%
dR = 0
→ Тренировка: объём +7%, вес +6.7%, паузы без изменений
```

**Педагогический смысл:** Тренер увеличивает и объём, и интенсивность равномерно. Нагрузка распределяется между параметрами — ни один не уходит в экстремальные значения. Наиболее «безопасная» стратегия.

**Когда применяется:**
- Универсальная стратегия для всех зон и периодов
- Когда нет специальной цели (объём или интенсивность)
- Зона 1 (good) — основная стратегия для восстановительных сессий

---

## 5. variative (Вариативная)

**Что меняется:** все три параметра (dV, dP, dR) пропорционально β и стандартным отклонениям (σ) из обучающих данных.

**Ограничения:** dV, dP, dR пропорциональны β_i × σ_i

**Формула:**
```
dV = ΔPC1 × β_V × σ_V / (β_V²σ_V² + β_P²σ_P² + β_R²σ_R²)  [и аналогично для dP, dR]
```

**Пример для Зоны 3 (ΔPC1 = +1.04):**
```
dV = +10.0%
dP = +9.9%
dR = +18.8%
→ Тренировка: объём +10%, вес +9.9%, паузы +18.8% (удлинены!)
```

**Педагогический смысл:** Мультипараметрический стимул. Тренер одновременно увеличивает объём, интенсивность и паузы. Удлинение пауз при росте нагрузки — это компенсаторный механизм, позволяющий «выдержать» более тяжёлую работу.

**Когда применяется:**
- Универсальная стратегия (good для всех зон)
- Развивающие тренировки с полным восстановлением между подходами
- Когда нужно максимально разнообразить стимул

| variative | good | good | good | good |

- **good** — дельты в пределах ±2σ от обучающей выборки (576 сессий)
- **caution** — дельты в пределах ±3σ, но за ±2σ
- **avoid** — дельты за ±3σ (запрещено использовать в микроциклах)

**balanced** и **variative** — единственные стратегии с good для всех 4 зон. Поэтому Зона 1 в микроциклах строится только через эти две стратегии.

## Как стратегии используются в микроциклах

Каждая сессия микроцикла = одна зона + одна стратегия. Тип микроцикла определяет **какие зоны**, а шаблон определяет **какие стратегии**:

```
Shock-микроцикл v1 (3 сессии):
  Сессия 0: Зона 4 + volume_only  → V +22.7%         (ударный пик)
  Сессия 1: Зона 1 + balanced     → V −16.7%, P −16.1% (компенсация)
  Сессия 2: Зона 3 + volume_only  → V +13.9%          (развивающая)
```

Один и тот же shock-микроцикл v2 использует другие стратегии:
```
  Сессия 0: Зона 4 + balanced     → V +11.4%, P +10.9% (ударный пик)
  Сессия 1: Зона 1 + variative    → V −12.5%, P −12.4%, R +23.6% (компенсация)
  Сессия 2: Зона 3 + balanced     → V +7.0%, P +6.7%   (развивающая)
```

Зоны одинаковые (4→1→3), стратегии разные → разные тренировочные воздействия с одинаковым биохимическим откликом.
